Logic Conditionals

  • This logical conditions that allow you to control the flow of a program.
  • if Statement
    • const age = 23;
    if (age < 18) {
    console.log('Sorry, you are not old enough to play this game');
    }
  • else Statement
    • const n = 12;
    if (n%2 === 0) {
    console.log('n is an even number');
    } else {
    console.log('n is an odd number');
    }
  • else if Statement
    • if (number === 4) {
        console.log('You rolled a four');
        } else if (number === 5) {
        console.log('You rolled a five');
        } else if(number === 6){
        console.log('You rolled a six');
        } else {
        console.log('You rolled a number less than four');
        }
  • Switch Statement
    • switch (number) {
        case 4:
        console.log('You rolled a four');
        break;
        case 5:
        console.log('You rolled a five');
        break;
        case 6:
        console.log('You rolled a six');
        break;
        default:
        console.log('You rolled a number less than four');
        break;
    }

Loops

  • Loops will repeat a pice of code until change the conditions.
  • While Loops
    • let bottles = 10;
    while (bottles > 0){
        console.log(`There were ${bottles} green bottles, hanging on a wall. And if one green bottle should accidentally fall, there'd be ${bottles-1} green bottles hanging on the wall`);
        bottles--;
    }
  • do .. while Loops
    • do {
    do something
} while(condition)
  • For Loops
    • for (initialization ; condition ; after) { do something }
